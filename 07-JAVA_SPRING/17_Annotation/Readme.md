## Java 에서 Annotation 은 어떤 기능을 하나요?

### 어노테이션의 기본 사용

1. 컴파일러에 정보 제공

   `추가 정보를 제공하여 특정 부분의 코드에 대한 경고나 오류를 발생시킬 수 있다`

   ```java
   @Override
       public void someMethod() {
           // method implementation
       }
   ```

2. 컴파일 단계 혹은 배포 단계에서의 처리

   `소프트웨어 도구는 주석 정보를 처리하여 코드, XML 파일 등을 생성할 수 있다`

3. 런타임시에 처리

   `런타임 시 검사할 수 있다`
